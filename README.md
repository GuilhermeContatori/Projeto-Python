# 🐍 Automação de Cadastro de Produtos

Este projeto em Python realiza a automação do cadastro de produtos em um sistema web de uma empresa.

## 🚀 Funcionalidades

- Leitura de dados a partir de um arquivo CSV
- Preenchimento automático de formulários
- Navegação entre campos utilizando teclado
- Cadastro de múltiplos produtos de forma automatizada

## 🛠️ Tecnologias utilizadas

- Python
- PyAutoGUI
- Pandas

## 📁 Estrutura do projeto

- `main.py` → Script principal de automação
- `produtos.csv` → Base de dados com os produtos

## ⚙️ Como funciona

O script lê os dados do arquivo CSV e utiliza automação de interface para:

1. Preencher os campos do sistema
2. Navegar entre os campos com TAB
3. Inserir informações como:
   - Código
   - Marca
   - Tipo
   - Categoria
   - Preço
   - Custo
   - Observações

## 💡 Observações

- O projeto utiliza coordenadas fixas da tela (x, y)
- Pode ser necessário ajustar dependendo da resolução do monitor
- Ideal para tarefas repetitivas de cadastro

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar automação com Python e manipulação de dados, simulando um cenário real de uso em empresas.
